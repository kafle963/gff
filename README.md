# Happy Birthday, Krishpa 🌸

An interactive, responsive birthday celebration webpage.

## Features
- **Envelope opening** with falling confetti.
- **Birthday Cake** with interactive candles that you can blow out (tap to extinguish) with sound feedback.
- **Typewriter Love Letter** with realistic typing animation.
- **Interactive Birthday Wishes** grid containing gift card-like popups.
- **Heart Catching Game** with scoring system, burst effects, and sound feedback.
- **Grand Finale Fireworks** with a custom canvas particle system (tap/click to shoot rockets!).
- **Custom Cursor Trail** and cherry blossom floating petals.
- **Procedural Chime Music** toggleable via a button.

## How to Run
Simply double-click or open [index.html](index.html) in any modern web browser.
